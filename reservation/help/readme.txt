Используемая БД: MS Sql server express 2017

Как должны быть созданы таблицы показано на скриншотах reserv.jpg и products.jpg
Скрипты заполнения таблиц:reserv.txt и products.txt
Перед тестированием нужно настроить файл app.config на ваши параметры подключения
